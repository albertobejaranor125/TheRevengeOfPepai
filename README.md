# The Revenge of Pepai

The Revenge of Pepai is a 2D arcade game developed in Unity and playable in the browser through WebGL.

Pepai is a rebellious robot trying to repair every broken machine he finds while searching for his place and inner peace.

The goal is simple: repair as many machines as possible before time runs out. Players can sprint, dash, jump, and chain combos to increase their score.

## Features

- Fast-paced arcade gameplay
- Sprint, dash, jump, and double jump movement system
- Combo-based score system
- Machine spawning logic with random positions
- Timer and game over flow
- Sound effects and music integration
- WebGL browser build

## Controls

- Move: A / D or Left Stick
- Sprint: Left Shift or Left Stick Press
- Jump: Space or South Button
- Dash: Left Mouse Button, O Key, or Right Shoulder Button

## Technologies Used

- Unity
- C#
- WebGL
- TextMeshPro
- Unity Input System

## Development

The main focus during development was improving game feel through movement, dash responsiveness, machine repair feedback, score combos, and quick arcade pacing.
